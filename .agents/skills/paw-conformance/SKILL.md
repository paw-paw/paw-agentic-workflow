# paw-conformance

Status: Antigravity candidate physical file. This replaces handoff-local
`paw-verify`; `paw-verify` is not a live portable operation.

Load `paw/orchestration/conformance.md`, the governing rule set, and existing
checks or manual evidence.

Do: classify automated checks, manual evidence, deferred checks, blocked rules,
and accepted gaps.

Do not: invent rules from tests or claim stable runtime support.

Output: conformance chain, evidence gaps, and follow-up.
