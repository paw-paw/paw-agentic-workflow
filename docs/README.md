# Documentation

This directory contains the minimum public governance for the PAW pre-alpha bootstrap.

## Precedence during bootstrap

1. This `docs/README.md` for the public documentation map.
2. `AGENTS.md` for repository-wide operating rules.
3. The final handoff for the active `paw-foundation` phase, when available locally under ignored `_inbox/final/`.
4. Governed live artifacts created by the active patch.
5. The inherited v1 bootstrap under `sdd/**` and `.codex/**`.
6. `_inbox/decision_ledger.md` only for exceptional clarification.

Private `_inbox/` material is operational input, not public repository content. Legacy research is not copied into the normal working tree.

## Documents

- `governance/BOOTSTRAP-STATUS.md`: scope and limitations of the public seed.
- `governance/PROVENANCE.md`: source repository and extraction record.
- `licensing/OUTPUT-POLICY.md`: initial policy for user outputs, templates, and assets.
- `provenance/public-seed-imports.tsv`: byte-identical imports and classifications.
- `provenance/public-seed-exclusions.tsv`: excluded source surfaces and reasons.
